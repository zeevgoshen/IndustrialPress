# AI prompt documentation (mandatory)

Log every AI-assisted prompt used during this assignment.

## File naming

```
prompts/YYYY-MM-DD-short-description.md
```

Example: `prompts/2026-05-24-docker-compose-skeleton.md`

## Template

```markdown
# Title

**Date:** YYYY-MM-DD  
**Tool:** Cursor / Copilot / etc.  
**Goal:** What you were trying to accomplish

## Prompt

(paste your prompt)

## Outcome

- What changed
- Files touched
- Follow-up edits you made manually
```

## Rules

- Document prompts **as you go** — do not batch at the end.
- Include failed attempts if they shaped the final design.
- Do not paste secrets or credentials.
